Data Explorer App (Streamlit)

Project Description
This project is a Streamlit web application that allows users to explore datasets using statistical analysis and visualization.

Users can upload a CSV file, select numeric columns, and view important statistics like mean, median, standard deviation, and data distribution.

Features
- Upload CSV dataset
- Preview dataset
- Select numeric column
- View statistics:
  - Mean
  - Median
  - Standard Deviation
  - Minimum & Maximum
- Histogram visualization
- Write and display insights

Technologies Used
- Python
- Streamlit
- Pandas
- Matplotlib


How to Run
1. Install dependencies:
 pip install -r requirements.txt

2. Run the app:
streamlit run app.py

 Concepts Used
1. Mean
Average value of data

2. Median
Middle value when data is sorted

3. Standard Deviation
Measures how spread out the data is

4. Histogram
Graph showing frequency distribution
